# Netflix_Customer_Churn

Welcome to the **Netflix Project** repository!
This project analyzes customer behaviour on Netflix to understand why some subsscribers cancel their subscriptions. Using data analysis and predictive modeling echniques, the study identifies patters and key factors influencing chrun.

## Project Overview

This project involves:
1. **Data Cleaning:** Cleaning the Netflix data and assigning names to numeric datasets which can prove complex to analyze
2. **Analytical & Reporting:** Creating Excel-based reports and dashboard for actionable insights.

## Project Objectives
1. Analyze customer behaviour to understand patterns that lead to subscription cancellations.
2. Identify key factors influencing churn, such as pricing, content availability, or viewing habits.
3. Provide insights and recommendations to improve customer retention strategies.
4. Support data-driven decision making for reducing churn and increasing customer satisfction.
